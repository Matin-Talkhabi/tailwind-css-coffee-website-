# Coffee Shop Website

A modern, visually appealing website for a coffee shop, featuring a responsive design, custom fonts, and engaging images. This project is ideal for showcasing products, blog posts, and information about a coffee shop or café.

## Features

- Responsive and mobile-friendly layout
- Custom Persian and English fonts
- Product and blog image galleries
- SVG icons and illustrations
- Organized folder structure for easy maintenance
- Ready-to-use static HTML, CSS, and JavaScript files

## Folder Structure

```
public/
│
├── font/         # Custom fonts (Dana, Morabba, Vazirmatn)
├── img/          # Images and SVGs (products, blogs, categories, icons)
├── script/       # JavaScript files (main script and Swiper.js)
├── style/        # CSS files (main styles and Swiper styles)
└── index.html    # Main HTML file
```

## Getting Started

1. **Clone or Download** this repository.
2. Open `public/index.html` in your web browser to view the website.
3. All assets (fonts, images, scripts, styles) are included locally—no build process required.

## Customization

- Replace images in the `img/` folder to update products, blogs, or branding.
- Edit `index.html` to change content, sections, or layout.
- Modify styles in `style/final.css` for custom colors, fonts, or spacing.
- Update or add JavaScript in `script/script.js` as needed.
